生成SSH Key：注意C是大写。

当看到以下界面时，表示Key生成成功。

配置Key

仔细看刚才输出的字符信息，找到id_rsa.pub文件的对应目录，用记事本等工具打开，拷贝文件中内容。
登陆github，选择yourname.github.io这个Repository，选择左边Deploy Key，选择右边Add deploy key，将刚才复制的id_rsa.pub中的内容粘贴到Key中，勾选Allow write access，然后点击Add Key。
